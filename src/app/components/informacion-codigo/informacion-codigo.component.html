Frontend con Angular

Angular gestiona la interfaz del usuario con formularios reactivos y servicios para enviar datos al backend. Su estructura modular facilita la creación de aplicaciones dinámicas y responsivas.

Backend con Spring Boot

Spring Boot recibe y procesa la información enviada desde Angular, gestionando la lógica de negocio, acceso a datos y exponiendo APIs REST seguras y escalables.

Análisis con Weka

Weka aplica modelos de machine learning entrenados con datos médicos para predecir enfermedades. Su integración con Spring Boot permite análisis rápidos y precisos.
