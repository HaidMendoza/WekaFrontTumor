¿Cómo funciona Weka para detectar tumores?

Weka permite cargar conjuntos de datos médicos con características como edad, tamaño del tumor, textura, entre otros. Luego aplica algoritmos como árboles de decisión (J48) para entrenar modelos que predicen si un paciente podría tener un tumor maligno o benigno.
